# Park Transform

Implements the Park transformation to convert (α, β) stationary components 
into (d, q) rotating reference frame components using θ.

## Files
- `Park_Transform.v` — Core module  
- `Park_Transform_TB.v` — Testbench  
- `TB_Results_Park_Transform.png` — Output waveform  

## Notes
Designed for integration with the Clarke module for FOC motor control.
